All tags have default CSS values, for example an a tag is:

color: (internal value);
text-decoration: underline;
cursor: auto;

All tags can have their default CSS values overwritten by adding CSS.

This is especially important when removing margins!!!

Tags that have default margins set:

body tag has margin:8px;

paragraph (p) has margin: 1em 0em;

header1 (h1) has margin: .67em 0em;

header2 (h2) has margin: .83em 0em;

header3 (h3) has margin: 1em 0em;

header4 (h4) has margin: 1.33em 0em;

header5 (h5) has margin: 1.67em 0em;
header6 (h6) has margin: 2.33em 0em;
  • unordered lists (ul) has margin:1em 0em; and padding-left:40px;
  1. ordered lists (ul) has margin:1em 0em; and padding-left:40px;
